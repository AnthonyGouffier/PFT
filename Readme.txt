Projet TFT feat Pokemon:
Le jeu est basé sur TeamFightTactic, un jeu de Riot Games. Le jeu met en scene des personnages (dans notre cas des pokémon) qui s'affronteront de façon autonomes.En fontion du résultat de cet affrontement le dresseur perdra progressivement ces PV. Le but est d'être le dernier dresseur en vie.
